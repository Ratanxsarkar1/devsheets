## 🚀 Features

- **Instant search** — filter by command name, description, or keyword
- **One-click copy** — copy any command to clipboard instantly
- **Plain-English explanations** — real-world scenarios, not dry manual text
- **Official docs links** — every command links directly to the source docs
- **Pure HTML/CSS/JS** — no framework, no build step, works offline
